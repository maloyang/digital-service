# 2019-03-21 建立網頁計畫開始

- 先用政府公開網的open data硬轉出GPS位置
- 把資料先入DB
- 用網頁呈現:
  - 框選區域以了解傷亡人數
  - 區域逐月的傷亡事件數趨勢圖
  - 查目前畫面範圍中的所有傷亡事件
